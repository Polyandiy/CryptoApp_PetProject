# CryptoApp_PetProject

Приложение с актуальной информацией о криптовалюте (загружает данные о ценах в реальном времени с бесплатного API (www.coingecko.com) и сохраняет в портфель текущего пользователя)

Использовались:
- SwiftUI
- Combine
- MVVM
- FileManager
- CoreData
- Animation
